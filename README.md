# OptimalGo Project


El Proyecto OptimalGo tiene como objetivo pronosticar los envíos de paquetería que se realizan entre particulares en las diferentes provincias españolas.

Con este fin, se calcularon 50 series temporales para cada una de las 50 provincias que dispone el territorio español. El modelo que mejor resultado obtuvo durante las pruebas fue [prophet], obteniendo un RMSE y MAPE optimo.
